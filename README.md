## ✨ **주요 기능**
- 💰 **수입/지출/고정자산 기록**  
- 📋 **입금여부, 견적서, 거래명세서, 적격증빙 관리**  
- 📆 **연도/월별 기록 조회**  
- 📊 **엑셀 파일로 데이터 내보내기**  

---

## ⚙️ **기술 스택**
- **Python 3.11**  
- **SQLite3** (가벼운 로컬 데이터베이스)  
- 📁 **외부 파일 관리:** 견적서 등은 `files` 폴더에 별도 저장  

---

## ⚠️ **유의사항**
- `files` 폴더 내의 파일은 **프로그램 외부에서 임의로 수정, 삭제, 이동하지 않는 것**을 권장합니다.  
- 데이터 무결성을 유지하기 위해 프로그램을 통한 관리가 필요합니다.
